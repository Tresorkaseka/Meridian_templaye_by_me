# Project Brief

## Overview
Develop a "Maximalist Refined" SaaS landing page. The application is built using React, TypeScript, and Tailwind CSS. The goal is to create a stunning, high-agency frontend interface that avoids generic AI visual biases.

## Key Requirements
- **Aesthetic Direction**: Maximalisme raffiné (refined maximalism). Features bold gradients and sculptural shadows.
- **Stack**: React, TypeScript (TSX), Tailwind CSS.
- **Design Overrides**: Must leverage `taste-skill` guidelines (e.g., DESIGN_VARIANCE: 8, MOTION_INTENSITY: 6, VISUAL_DENSITY: 4).
- **No Generics**: Avoid flat designs, generic card layouts, emojis, and default typography.
- **Images**: Utilize image generation for premium assets.
